# Контакт — десктоп-клієнт

Десктоп-застосунок (Electron) для CRM сервісного центру «Контакт»: облік
продажів, ремонтів і клієнтів, аналітика та друк чеків/актів. Це
клієнтська частина дипломного проєкту; вона працює з окремим бекендом на
.NET 8 Web API (репозиторій `Contact.API`) і спільною базою MySQL.

## Можливості

- **Продажі** — реєстрація з позиціями (POS), статуси, дублювання, друк чека.
- **Ремонти** — таблиця та канбан-дошка, статуси, акт виконаних робіт.
- **Клієнти** — картки, історія звернень, пошук, позначка клієнтів із Конфігуратора ПК.
- **Аналітика** — графіки доходу й розподілу за категоріями (Chart.js), вибір періоду.
- **Користувачі та ролі** — `superadmin` / `admin` / `master` з розмежуванням доступу.
- **Довідка** — пошук по розділах і гарячі клавіші (Esc, Ctrl+N, F5, Enter, `?`).

## Стек

Electron, ванільні JS/HTML/CSS, Chart.js. Без фреймворків — кожна сторінка
лежить у `renderer/features/<розділ>/` і користується спільними утилітами з
`renderer/core/` (тости, діалоги, телефони, автодоповнення, гарячі клавіші).

## Запуск

Потрібен Node.js.

```bash
npm install
npm start        # запуск у режимі розробки (electron .)
```

## Збірка інсталятора

```bash
npm run dist     # electron-builder, інсталятор NSIS для Windows
```

## Налаштування

Адреса бекенду задається в `renderer/core/config.js` (`window.API_BASE`).
Кнопка «Конфігуратор ПК» у шапці відкриває окремий веб-модуль конфігуратора
у системному браузері.

## Структура

```
main.js                 — головний процес Electron (вікно, відкриття посилань/PDF)
renderer/core/          — спільні стилі та утиліти (common.js, components.css, …)
renderer/features/      — сторінки: auth, dashboard, sales, repairs, clients,
                          analytics, users, settings, help, receipt
renderer/assets/        — іконки та логотипи
```
